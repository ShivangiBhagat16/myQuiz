<div>
	<h3 align="center">MY QUIZ</h3>
	- My Quiz is a single page application(SPA) where you can test your React learning skills. <br/>
	- There are total15 questions, and each question has points associated with it based on its difficulty level.<br/>
	- In order to complete the Quiz, you need to keep an eye on the global timer.<br/>
	- Your earned points with percentage will be displayed at the end of the Quiz.<br/>
	- Quiz can be restarted as many times as you like.

### Installation

1. Download the project
   ```sh
   git clone https://github.com/ShivangiBhagat16/myQuiz.git
   ```
2. To run project, first download node modules
   ```sh
   npm install
   ```
2. To run project, 
   ```sh
   npm start
   ```   
3. To load Questions API, Open new terminal and run
   ```sh
   npm run server
   ```
